# InnerHue Backend

Backend API for the InnerHue emotional wellness platform.

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

`` 
npm install
``

### Development

``
npm run dev
``

### Build

``
npm run build
``

## Project Structure

``
backend/
+-- src/
¦   +-- index.js          # Entry point
¦   +-- routes/           # API routes
¦   +-- controllers/       # Request handlers
¦   +-- models/           # Data models
¦   +-- middleware/       # Express middleware
¦   +-- utils/            # Utility functions
+-- package.json
``

## Environment Variables

Create a .env file in the root of the backend directory:

``
PORT=3001
NODE_ENV=development
``
